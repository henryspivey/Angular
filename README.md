# Angular
Ang-news is a  Reddit clone that uses AngularJS & Firebase. I learned how to develop CRUD (create, read, update, delete) services, securely authenticate users, access servers with $http, communicate with REST APIs using $resource, enable three-way data binding with your server, create directives for your user interface and more.
It is an implementation of <a href="https://thinkster.io/angulartutorial/learn-to-build-realtime-webapps/">this</a> Angular tutorial.
